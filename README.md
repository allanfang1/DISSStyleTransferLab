# StyleTransfer TP for UCBL DISS ImageMachineLearning Class

Project implementing image style transfer with PyTorch following paper by Gatys et al.: [Image Style Transfer Using Convolutional Neural Networks](https://www.cv-foundation.org/openaccess/content_cvpr_2016/html/Gatys_Image_Style_Transfer_CVPR_2016_paper.html). Optimize (SGD) the pixels of an image by basing the loss function on the latent space of pre-trained network VGG19.

- [peinture1_small.jpg](peinture1_small.jpg) — Style source image.
